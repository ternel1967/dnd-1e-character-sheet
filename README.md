# 1st Edition D&D Character Sheet Application

This repository contains a comprehensive character sheet application for 1st Edition Dungeons & Dragons. The application allows players to easily create, customize, and manage their characters while ensuring all essential statistics and details are maintained in one convenient location.

## Features
- ⚔️ Multi-character management with tabbed interface
- 📊 Complete ability scores with automatic modifiers
- 🎲 Interactive dice roller with history
- ✨ Spell management system with memorization tracking
- 📈 Auto-leveling based on experience points
- 🛡️ Combat statistics (THAC0, AC, HP tracking)
- 📚 Skill tracker with ability links
- 💰 Equipment and inventory management
- 💾 Local storage persistence
- 🖥️ Desktop app support via Electron

## Technologies Used
- **Frontend:** Vue 3, Pinia (state management)
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Desktop:** Electron
- **Rules Engine:** 1st Edition D&D rules
